Extras - Combate (Angular X React)

O objetivo foi realizar uma brincadeira comparativa entre as duas formas mais aplicadas atualmente de consumir dados via API utilizando JS.

Foi a primeira vez que usei react, o código não está completo do modo como eu gostaria, mas procurei estudar e aplicar para aprendizado.

Desse modo mesmo que de maneira simples pude ver os pontos negativos e positivos das duas perspectivas de desenvolvimento.

A aplicação em Angular procurei manter na estrutura padrão MVC.

As páginas foram desenvolvidas pensando de forma responsiva.
