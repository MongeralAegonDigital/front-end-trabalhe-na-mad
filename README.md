### O que foi usado ###

*  Pré processador SASS.
*  Task runner GulpJs.
*  Framework AngularJs.

###  Como rodar o projeto ###

*  Rodar no terminal - npm install.
*  Rodar no terminal - bower install.
*  Quando todas as dependências estiverem instaladas rodar o gulp no terminal.
